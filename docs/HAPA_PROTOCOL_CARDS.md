# Hapa Protocol Cards

Generated thumbnail art for complex Hapa protocols lives in:

`$HAPA_DESKTOP_ROOT/hapa/site/generated/protocol-cards`

## Series Intent

These images are lore-forward protocol thumbnails. Each card uses the same visual language while encoding the function of a protocol: parity, memory, sharpening, vectors, inter-node flow, agent delegation, avatar lineage, and Second Brain deployment.

The images intentionally avoid baked-in readable text so the app, wiki, or card renderer can overlay titles consistently.

## Operating Card Library

Use `site/generated/protocol-cards/protocol-card-library.json` for first-class Protocol Card records. This library points to:

- wiki card pages under `$HAPA_DESKTOP_ROOT/Hapa_Worldbuilding_Wiki/Cards/Protocol Cards/`;
- canonical protocol pages and Flow Explainers;
- Second Brain article slugs;
- card verbs and functions for app/game usage.

The current operating cards include the three sequential pass cards:

- `for-the-throat` - speed/deadline pass for minimum acceptable output with future-ready foundations.
- `every-which-way` - alternatives pass for local/remote, open/paid, append/replace decisions.
- `with-flair` - production-value pass for teaching, retention, and ecosystem wisdom.

They also include the primary Hapa protocol cards for parity, sovereign memory, sharpening, Love/Truth/Conviction, inter-node flow, Hermes delegation, avatar lineage, and Second Brain enrichment.
